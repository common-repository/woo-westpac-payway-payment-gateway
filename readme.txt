== Description ==

This plugin extends WooCommerce and provides a secure Iframe based checkout method using Westpac PayWay. The Iframe uses PayWay REST API and no credit card data is stored on your server. This means there is no need of PCI-DSS compliance so costs are reduced. This plugin works with AUD currency.
